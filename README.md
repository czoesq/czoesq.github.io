# Etsy Sales Dashboard

Interactive dashboard for analyzing Etsy sales performance.

## Features
- 📈 Monthly sales trends visualization
- 🏆 Top products performance tracking
- 💰 Price distribution analysis
- 🎨 Design category breakdown
- 📊 Real-time business insights

## Data Source
- 2,498 Etsy sales transactions (2025-2026)
- $68,217.69 total revenue analyzed
- 348 unique products tracked

## Access
Live at: https://czoesq.github.io/

## Privacy Note
This dashboard uses anonymized, aggregated sales data for business intelligence.
